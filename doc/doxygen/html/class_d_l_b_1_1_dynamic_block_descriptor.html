Detailed Description

Class describing distributed topology.

Provides useful methods for topology related prolems. Computes all necessary information, like block size and neighborhood based MPI rank and world size.

bool DBD::loadBalance ( PerfMeasure &  pm,
BlockData &  block 
)

Dynamic load balance method.

This method needs measured performance, which is send to master and evaluated. Re-partition and data migration may occur within this call based on measured values.

Upon true returned, balancing performed, BlockData will be updated Re-initialization of neighbor comms is necessary.

Parameters
pm- performance measurement object with data about last iterations
block- BlockData reference, will be update if load balance occurs
Returns
true if balancing performed
